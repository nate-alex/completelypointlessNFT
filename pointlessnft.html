This is a completely pointless NFT.

Well, at least it was when it was originally conceptualized. It all started with a mewny tweet talking about an NFT that was just the holder's public key on a black background. I spent about 30 minutes coding it up and adding a few pointless features including the count of each letter and number in the address, and a "decay" state which changes the background color slightly upon each transfer. One final quirk is that the NFT changes to show whichever address it currently resides in. Metadata may need to be refreshed on opensea for it to update quickly.

It was initially intended to be a meme, just something I coded out of boredom, and pricing was set to 0.042069 with the maximum cap set to 969. Shortly after release, I decided I would donate all the money from the primary sales to a charity (the actual charity is still TBD). I will link the tx hash of the donation here when it is complete. Maybe this project isn't completely pointless now.
